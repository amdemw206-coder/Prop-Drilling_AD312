## Project Title: AD 312 Refactoring Prop Drilling to Context
#### By: Amde Wubshet

### Breif Description
As applications grow, passing data through multiple layers of components—known as Prop Drilling—can make code difficult to maintain and debug. We will take a “messy” application where user settings are being passed manually through the component tree and refactor it to use the React Context API and the useContext hook.

### Commands 
After installing the repo, navigate into your project directory: *cd context-refactor-app*. Then start the development server: *npm run dev*.
